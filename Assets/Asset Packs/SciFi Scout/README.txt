Sci-Fi female scout with a drone
--------------------------------
Version 1.5 
--------------------------------
How to use:
Drop the prefabs in your scene

This package includes:
-1 character called "Scout" (3 LOD levels)
-1 drone
-13 character animations
-3 blend shapes on Scout/Face
-4 materials (mat_suit and mat_armor is for Scout, mat_drone is for drone)

Does this model supports mecanim?
Yes, the model is mecanim ready - humanoid skeleton. You can use third party animations with it. 

Source model is located in Source.zip
